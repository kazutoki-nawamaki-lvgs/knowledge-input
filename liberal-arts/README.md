# Liberal Arts

教養ノートを置く場所です。歴史、哲学、テクノロジーなどのテーマを、単なる用語説明ではなく、問いから始まる読み物として扱います。

扱うテーマは `topics/topics.csv` を参照します。カテゴリ列に合わせて、`liberal-arts/哲学/001_テーマ.md`、`liberal-arts/歴史/001_テーマ.md` のように保存します。

## 使い方

```sh
make liberal-arts
```

以前からのコマンド名として、同じ処理を次でも実行できます。

```sh
make daily
```

テーマを指定する場合:

```sh
make liberal-arts TOPIC="貨幣とは何か"
```

## 書き方の方針

- `templates/daily.md` を起点にし、問い、背景、具体例、現代への接続、持ち帰りが自然につながる本文にする
- 歴史や哲学の説明だけで終わらせず、読者が今の生活、仕事、ニュース、人間関係を見直せるようにする
- 具体例は複数を浅く並べず、理解を助ける代表例を1つ深めに扱う
- 長い章では `###` の小見出しを使い、見出しだけで論点の流れが追えるようにする
- 最後の問いは、読者が実際に観察、内省、比較へ移せる具体的な問いにする

教養ノートでは、**知識の量よりも、ものの見方が少し変わること**を重視します。
