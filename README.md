# 教養インプット

毎日5分で読める教養ノートを作るための小さなファイルベース運用です。

## 使い方

今日の日付で1本作る:

```sh
make daily
```

日付を指定して作る:

```sh
make daily DATE=2026-05-21
```

テーマを指定して作る:

```sh
make daily TOPIC="貨幣とは何か"
```

日付とテーマを指定して作る:

```sh
make daily DATE=2026-05-21 TOPIC="なぜ宗教はなくならないのか"
```

生成されるファイルは `daily/YYYY-MM-DD_テーマ.md` です。
同じ名前のファイルがすでにある場合は上書きせず、`_v2`, `_v3` のように別名で作ります。

## 構成

- `daily/`: 毎日の教養ノート
- `topics/topics.csv`: 自動選出用のテーマ一覧
- `templates/daily.md`: 生成されるノートのテンプレート
- `scripts/new_daily.py`: ノート生成スクリプト

## ノートの基本形

各ノートは次の流れで書きます。

- 今日の問い
- まず結論
- 背景知識
- 具体例
- 現代とのつながり
- 考える問い
- 関連キーワード
- 次に読むなら

最初は空欄を埋めるだけで十分です。慣れてきたら、読んだ本・記事・動画のメモを追記していくと、あとから見返せる知識ログになります。
